# Feature 042 — Waiting List Confirmation Email via AWS SES

## Initial Prompt

> "As a guest, in the landing page, I would like to be able to subscribe to a waiting list. The e-mail has to be sent from no-reply@popup-live.fr with AWS SES. This will serve during the AWS Summit in order to increase my chances to request to get the AWS Activate for startups."

---

## User Stories

| User Story | Status |
| :--------- | :----- |
| As a guest, on the LandingPage, when I submit the buyer waitlist form, I should receive a confirmation email from no-reply@popup-live.fr | completed |
| As a guest, on the LandingPage, when I submit the seller waitlist form, I should receive a confirmation email tailored to sellers from no-reply@popup-live.fr | completed |
| As a guest, when I try to subscribe with an email already on the list, I should see an error message and not receive a duplicate email | completed |
