# Feature 029 — Live Form Redesign

**Initial prompt:**

> As a seller, in the lives page, when programming a new live, I want to see the design shown in the screenshot. I'm referring to the card component for the product to associate to the live, including the font style, background color, checkbox and the different inputs styling. I'm also referring to the order of the elements displayed.

---

## User Stories

| User Story                                                                                                        | Status  |
| :---------------------------------------------------------------------------------------------------------------- | :------ |
| As a seller, in the "+ Live" form, fields appear in order: title → date/time → description → cover → products     | planned |
| As a seller, the date and time inputs are displayed as pill buttons showing a human-readable value                | planned |
| As a seller, product cards display with a dark background, large image, Syne bold name, and primary-colored price | planned |
| As a seller, the checkbox turns yellow/primary when a product is selected                                         | planned |
| As a seller, a "Sélectionner" button in the products header toggles selecting all products                        | planned |
