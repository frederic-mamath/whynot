# Feature 028 — Live Cover Image

**Initial prompt:**

> As a seller, when I program a live, I can select a picture for the background displayed in the preview.

---

## User Stories

| User Story                                                                                                                          | Status  |
| :---------------------------------------------------------------------------------------------------------------------------------- | :------ |
| As a seller, in the live scheduling form, I can upload a cover image for my live                                                    | planned |
| As a seller, in the live scheduling form, I see an immediate preview of the selected cover image                                    | planned |
| As a seller, in the edit live dialog, I can change the cover image of an existing scheduled live                                    | planned |
| As a user, on the home page highlight block, the live's cover image is used as background (falls back to the host's avatar if none) | planned |
