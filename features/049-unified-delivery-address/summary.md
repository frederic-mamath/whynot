# 049 — Unified Delivery Address

## Initial Prompt

> We have two different cards to set the delivery address. I would like to have only one with two options: manually typing your address, or picking a Point Relais with Mondial Relay on a map. When pressing a button, a dialog opens. The Mondial Relay dialog uses Leaflet + OpenStreetMap (free, no subscription). Max 1 address at a time (0 or 1).
