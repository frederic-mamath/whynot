# Feature 037 — Payment Method Management

**Initial prompt:**

As a buyer, when I setup my payment method, I want to also be able to setup a credit card or visa card. After the setup, I should be able to edit or remove my payment method.

---

## User Stories

| User Story                                                                                                                              | Status  |
| :-------------------------------------------------------------------------------------------------------------------------------------- | :------ |
| As a buyer, in the profile page, when I open the payment setup dialog, I should see a card input form (in addition to Google/Apple Pay) | planned |
| As a buyer, in the profile page, when I fill in my card details and submit, I should see my card saved in my payment methods            | planned |
| As a buyer, in the profile page, when I have a saved payment method, I should see a delete button next to it                            | planned |
| As a buyer, in the profile page, when I click the delete button on a payment method, it should be removed from my account               | planned |
| As a buyer, in the profile page, when I click "Modifier", I should be able to add a new payment method                                  | planned |
