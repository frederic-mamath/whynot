# Feature 040 — Seller Hub "Ma boutique"

## Initial Prompt

> As a seller, when I click on "Vendre", I see the main menu for sellers. It is composed of 4 cards in the order of the seller flow:
> 1. Inventaire, to build the shop by adding products
> 2. Lives, to program the lives
> 3. Livraisons, to followup with products to ship, their status and tracking packages
> 4. Analytics, to withdraw the money from sells that will be available after the package have been received

## User Stories

| User Story | Status |
| :--------- | :----- |
| As a seller, when I tap "Vendre" in the BottomNav, I should land on the "Ma boutique" hub | completed |
| As a seller, in the hub, I should see 4 navigation cards in workflow order | completed |
| As a seller, each card shows a live count so I know at a glance what needs attention | completed |
| As a seller, the Livraisons card highlights the count when orders are pending | completed |
| As a seller, tapping a card navigates me to the correct section | completed |
| As a seller, each sub-page has a back arrow returning me to the hub | completed |
| As a seller, the BottomNav "Vendre" tab stays active across all seller sub-pages | completed |
| As a seller, the Analytics card shows a coming-soon page until the feature is built | completed |
| As a seller, in the Inventaire page, I should see products as vertical cards with a large image | completed |
| As a seller, in the Inventaire page, I can edit a product (navigates to edit page) | completed |
| As a seller, in the Inventaire page, I can delete a product with a confirmation dialog | completed |
| As a seller, in the Inventaire page, I should see an inline add button instead of a FAB | completed |
