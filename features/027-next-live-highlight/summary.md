# Feature 027 — Next Live Highlight

**Initial prompt:**

> As a user, in the home page, I would like to see the next incoming live in highlight. There should be a countdown. The button "Partager" should open the sharing panel on iOS. The background image is the live's author avatar.

---

## User Stories

| User Story                                                                                                                  | Status  |
| :-------------------------------------------------------------------------------------------------------------------------- | :------ |
| As a user, on the home page, I should see a highlight block showing the next scheduled live                                 | planned |
| As a user, on the home page, I should see a real-time countdown (days/hours/min/sec) until the next live starts             | planned |
| As a user, on the home page, I should see the live's title and description in the highlight block                           | planned |
| As a user, on the home page, I should see the host's avatar as the background of the highlight block                        | planned |
| As a user, on the home page, when I click "Me rappeler", a .ics calendar event is downloaded so I can add it to my calendar | planned |
| As a user, on the home page, when I click "Partager", the iOS/browser share sheet opens with the live's URL                 | planned |
| As a user, on the home page, when no live is scheduled, I should see an empty placeholder block                             | planned |
