# 045 — Lives Discovery

## Initial Prompt

> I would like a similar behaviour for the list of lives in the HomePage. I would like to see only 4 lives ordered by starts_at DESC. Lives that started more than 3 hours ago should be hidden. The "Voir tout" button should appear only if there are 5 or more active lives, and navigate to a new /lives page. The /lives page should show active and future lives with category filters (multi-select, OR logic) and a sort toggle between title ASC and starts_at ASC.
