# Feature 030 — Active Lives Discovery

**Initial prompt:**

> As a SELLER, in on-going live where i'm not the host, I can participate as a BUYER.

---

## User Stories

| User Story                                                                                                            | Status  |
| :-------------------------------------------------------------------------------------------------------------------- | :------ |
| As a seller, on the home page, I can see a section "En direct maintenant" listing all active lives from other sellers | planned |
| As a seller, on the home page, when I click "Rejoindre" on an active live card, I join it as a viewer (buyer mode)    | planned |
| As a user, on the home page, the "En direct" section is hidden when no live is currently active                       | planned |
