# 047 — Placeholder Component

## Initial Prompt

> I would like to have a consistent design for placeholders. In the ProfilePage, there are two different designs to add entities such as payment cards and delivery addresses. Create a shared `Placeholder` component (Icon: React.ReactNode, title: string, ButtonListProps?: ButtonV2Props[]) and migrate all 17 empty state patterns found across 13 files to use it.
